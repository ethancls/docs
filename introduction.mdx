## Le Web Sémantique

Le Web Sémantique est une extension du Web actuel qui permet de structurer les données de manière à ce qu'elles soient compréhensibles par les machines. Il utilise des standards tels que RDF (Resource Description Framework) et OWL (Web Ontology Language) pour représenter les informations de manière formelle.
